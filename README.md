## ☁️ Deskripsi Sistem

Sistem Informasi Mahasiswa adalah aplikasi berbasis web yang dibangun menggunakan framework Laravel dengan konsep MVC (Model-View-Controller). Sistem ini dirancang untuk mengelola data mahasiswa secara digital, terstruktur, dan efisien.

Aplikasi ini menggunakan database cloud PostgreSQL (Supabase) sehingga data dapat disimpan dan diakses secara online. Selain itu, sistem ini juga menerapkan konsep deployment agar aplikasi dapat diakses melalui internet.

Frontend aplikasi dibangun menggunakan Blade Template Laravel dengan bantuan Bootstrap untuk menciptakan tampilan yang sederhana, responsif, dan mudah digunakan.

Project ini juga dikembangkan secara kolaboratif menggunakan GitHub dengan pembagian tugas berdasarkan peran masing-masing anggota tim.
